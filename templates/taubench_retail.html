TAU-bench Retail

TAU-bench is a benchmark for Tool-Agent-User Interaction in Real-World Domains. TAU-bench Retail evaluates AI agents on tasks in the retail shopping domain, such as cancelling orders, address changes, and checking order status.

Paper: τ-bench: A Benchmark for Tool-Agent-User Interaction in Real-World Domains (Yao et al., 2024)

Changelog: Removal of TAU-bench Few Shot Agent (December 8, 2025)

We have removed the TAU-bench Few Shot agent results from the leaderboard due to a critical data leakage issue. The Few Shot scaffold contained test set examples in its demonstration file, compromising evaluation integrity. For technical details, see Section 4.2 and Appendix A5 of the HAL paper. All results now use the valid TAU-bench Tool Calling agent.
